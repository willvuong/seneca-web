What is HTML?

HTML stands for (H)yper(T)ext (M)arkup (L)anguage. It is a set of markup symbols or codes inserted in a file intended to be displayed on a World Wide Web browser page such as Google Chrome or Internet explorer.

- Hypertext is text with hyperlinks.

- Markup tells the web browser how to display a web page's words and images for the user.

The markup symbols/indicators are called tages which are enclosed in angle brackets.

- Within these tags, there are elements, which is the content within the tags.

Example of a Simple HTML document:

<html>

<head>

   <title>WEB222</title>

</head>

<body>

   <h1>Basic HTML Document Structure</h1>

   <p>This is a paragraph.</p>

   <p>Here are links to

     <a href="https://ict.senecacollege.ca/">School of ICT</a>

     and

     <a href="http://www.senecacollege.ca/">Seneca College.</a>

  </p>

</body>

</html>
